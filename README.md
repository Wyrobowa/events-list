# React Event Form

App based on React and Redux, migrated from Create React App to [Vite](https://vitejs.dev/).

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode using Vite.<br />
Open [http://localhost:5173](http://localhost:5173) (default Vite port) to view it in the browser.

The page will reload if you make edits.

### `npm test`

Launches the test runner using [Vitest](https://vitest.dev/).

### `npm run build`

Builds the app for production to the `dist` folder.<br />
It correctly bundles React in production mode and optimizes the build for the best performance.

### `npm run serve`

Serves the production build from the `dist` folder for preview.

### `npm run lint`

Runs ESLint to check the code for style and potential errors.

## Tech Stack

*   **Framework**: React 19
*   **Build Tool**: Vite
*   **Language**: TypeScript
*   **Design System**: Tharaday DS
*   **State Management**: Redux, React-Redux, Redux Toolkit
*   **Routing**: React-Router-Dom v7
*   **Validation**: Yup
*   **Testing**: Vitest, React Testing Library
*   **Date Handling**: Day.js
